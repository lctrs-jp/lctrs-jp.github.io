Crisys2cdcc : a proof system cDCC for all-path reachability problems in Crisys2

Downloads
  • crisys2cdcc (OCaml bytecode)
  • crisys2cdcc.sh (shell script for automatic mode)

Requirement

  • The tool is implemented by OCaml (ver. 5.1.0), and to run the downloaded bytecode, ocaml (or later version) is necessary.
  • The tool needs an SMT solver that follows SMT-LIB 2 format. Please put the following file crisys2.config into the same directory as the tool: 
    (SMTSOLVER STDIN "xxxx")
(VERBOSE on)
    Please replace xxxx by a command for the SMT solver you use. For example, please write (SMTSOLVER STDIN "z3 -in") for Z3.

How to Use

Run crisys2cdcc from Command Line via the shell script crisys2cdcc.sh as follows: 

> ./crisys2cdcc.sh rw1_rf.lctrs2
