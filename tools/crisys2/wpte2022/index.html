Crisys2cdcc : a proof system DCC- for All-Path Reachability problems in Crisys2

Journal Submission version

This page provides a tool Crisys2cdcc, a proof system DCC- for All-Path Reachability problems in Crisys2. Crisys2, Constrained Rewriting Induction SYStem (version 2), is a proof system based on Constrained Rewriting Induction [KN14,FKN17] for logically constrained term rewrite systems (LCTRSs) [KN13], the current version of which works interactively. cDCC [KN23] is a simplified variant of DCC [CL18], a proof system for all-path reachability (APR) problems of LCTRSs.

Requirement

  • The tool is implemented by OCaml (ver. 4.12.1), and to run the downloaded bytecode, ocaml (or later version) is necessary.
  • The tool needs an SMT solver that follows SMT-LIB 2 format. Please put the following file crisys2.config into the same directory as the tool: 
    (SMTSOLVER STDIN "xxxx")
(VERBOSE on)
    Please replace xxxx by a command for the SMT solver you use. For example, please write (SMTSOLVER STDIN "z3 -in") for Z3.

How to Use

Run crisys2cdcc from Command Line as follows: 

> ./crisys2cdcc example_2-24_rf.rmt
or 
> ocamlrun ./crisys2cdcc example_2-24_rf.rmt

The tool tries to prove a problem written the input file, constructing a proof tree. The search space for proof trees is limited for the height, and the upper limit is X * Y from [X,Y] in prove [X,Y] in main : apr;. For example, the limit is 100 for [10,10].

Input Files

The tool takes an LCTRS written in a simplified varaint of RMT format. RMT, Rewriting Modulo Theories Tool, is a proof system for APR of LCTRSs (see also ISR 2019 Lecture by Stefan Ciobaca). We restrict the format to asynchronous integer transition systems with shared variables [KN23padl] which are written as LCTRSs.

sorts Process, Semaphore, Cnfg, Fcall;

signature
  cnfg    : Process Process Process Semaphore Int Int -> Cnfg,
  p       : Fcall Int -> Process,
  sem     : Int Int Int -> Semaphore,
  rdr7    : -> Fcall,
  rdr9    : Int -> Fcall,
  rdr10   : Int -> Fcall,
  rdr11   : Int -> Fcall,
  wrtr18  : -> Fcall,
  wrtr19  : -> Fcall,
  wrtr20  : -> Fcall,
  success : -> Cnfg,
  error   : -> Cnfg;

variables
  P1 : Process, P2 : Process, P3 : Process,
  N : Int, N1 : Int, N2 : Int, N3 : Int,
  S : Int, D : Int, T : Int, S1 : Int, D2 : Int, T2 : Int,
  X : Int, Y : Int, A : Int;

constrained-rewrite-system main
  (cnfg (p rdr7      0) P2 P3 (sem S D T) X 0)                                                                   => (cnfg (p (rdr9  0) 0) P2 P3 (sem S  D  T ) X 0),
  (cnfg (p (rdr9 Y)  0) P2 P3 (sem S D T) X 0) /\ (band (bnot (mequals S 0)) (mequals S1 (mminus S 1)))          => (cnfg (p (rdr10 Y) 0) P2 P3 (sem S1 D  T ) X 0),
  (cnfg (p (rdr9 Y)  0) P2 P3 (sem S D T) X 0) /\ (band (mequals S 0) (mequals T2 (mplus T 2)))                  => (cnfg (p (rdr9  Y) T) P2 P3 (sem S  D  T2) X 0),
  (cnfg (p (rdr9 Y)  N) P2 P3 (sem S D T) X 1) /\ (band (mequals N D) (bnot (mequals N 0)))                      => (cnfg (p (rdr10 Y) 0) P2 P3 (sem S  D  T ) X 0),
  (cnfg (p (rdr10 Y) 0) P2 P3 (sem S D T) X 0)                                                                   => (cnfg (p (rdr11 X) 0) P2 P3 (sem S  D  T ) X 0),
  (cnfg (p (rdr11 Y) 0) P2 P3 (sem S D T) X 0) /\ (band (bnot (mequals T (mplus D 2))) (mequals D2 (mplus D 2))) => (cnfg (p rdr7      0) P2 P3 (sem S  D2 T ) X 1),
  (cnfg (p (rdr11 Y) 0) P2 P3 (sem S D T) X 0) /\ (band (mequals T (mplus D 2)) (mequals S1 (mplus S 1)))        => (cnfg (p rdr7      0) P2 P3 (sem S1 D2 T ) X 0),

  (cnfg P1 (p rdr7      0) P3 (sem S D T) X 0)                                                                   => (cnfg P1 (p (rdr9  0) 0) P3 (sem S  D  T ) X 0),
  (cnfg P1 (p (rdr9 Y)  0) P3 (sem S D T) X 0) /\ (band (bnot (mequals S 0)) (mequals S1 (mminus S 1)))          => (cnfg P1 (p (rdr10 Y) 0) P3 (sem S1 D  T ) X 0),
  (cnfg P1 (p (rdr9 Y)  0) P3 (sem S D T) X 0) /\ (band (mequals S 0) (mequals T2 (mplus T 2)))                  => (cnfg P1 (p (rdr9  Y) T) P3 (sem S  D  T2) X 0),
  (cnfg P1 (p (rdr9 Y)  N) P3 (sem S D T) X 1) /\ (band (mequals N D) (bnot (mequals N 0)))                      => (cnfg P1 (p (rdr10 Y) 0) P3 (sem S  D  T ) X 0),
  (cnfg P1 (p (rdr10 Y) 0) P3 (sem S D T) X 0)                                                                   => (cnfg P1 (p (rdr11 X) 0) P3 (sem S  D  T ) X 0),
  (cnfg P1 (p (rdr11 Y) 0) P3 (sem S D T) X 0) /\ (band (bnot (mequals T (mplus D 2))) (mequals D2 (mplus D 2))) => (cnfg P1 (p rdr7      0) P3 (sem S  D2 T ) X 1),
  (cnfg P1 (p (rdr11 Y) 0) P3 (sem S D T) X 0) /\ (band (mequals T (mplus D 2)) (mequals S1 (mplus S 1)))        => (cnfg P1 (p rdr7      0) P3 (sem S1 D2 T ) X 0),

  (cnfg P1 P2 (p wrtr18 0) (sem S D T) X 0) /\ (band (bnot (mequals S 0)) (mequals S1 (mminus S 1)))          => (cnfg P1 P2 (p wrtr19 0) (sem S1 D  T ) X 0),
  (cnfg P1 P2 (p wrtr18 0) (sem S D T) X 0) /\ (band (mequals S 0) (mequals T2 (mplus T 2)))                  => (cnfg P1 P2 (p wrtr18 T) (sem S  D  T2) X 0),
  (cnfg P1 P2 (p wrtr18 N) (sem S D T) X 1) /\ (band (mequals N D) (bnot (mequals N 0)))                      => (cnfg P1 P2 (p wrtr19 0) (sem S  D  T ) X 0),
  (cnfg P1 P2 (p wrtr19 0) (sem S D T) X 0)                                                                   => (cnfg P1 P2 (p wrtr20 0) (sem S  D  T ) 1 0),
  (cnfg P1 P2 (p wrtr20 0) (sem S D T) X 0) /\ (band (bnot (mequals T (mplus D 2))) (mequals D2 (mplus D 2))) => (cnfg P1 P2 (p wrtr18 0) (sem S  D2 T ) X 1),
  (cnfg P1 P2 (p wrtr20 0) (sem S D T) X 0) /\ (band (mequals T (mplus D 2)) (mequals S1 (mplus S 1)))        => (cnfg P1 P2 (p wrtr18 0) (sem S1 D  T ) X 0),

  (cnfg P1 P2 P3 (sem S D T) X A) => success,

  (cnfg (p (rdr10 Y) N1) (p (rdr10 Y) N2) (p wrtr19 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr11 Y) N1) (p (rdr10 Y) N2) (p wrtr19 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr10 Y) N1) (p (rdr11 Y) N2) (p wrtr19 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr11 Y) N1) (p (rdr11 Y) N2) (p wrtr19 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr10 Y) N1) (p (rdr10 Y) N2) (p wrtr20 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr11 Y) N1) (p (rdr10 Y) N2) (p wrtr20 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr10 Y) N1) (p (rdr11 Y) N2) (p wrtr20 N3) (sem S D T) X A) => error,
  (cnfg (p (rdr11 Y) N1) (p (rdr11 Y) N2) (p wrtr20 N3) (sem S D T) X A) => error;

rewrite-system apr
  (cnfg (p rdr7 0) (p rdr7 0) (p wrtr18 0) (sem 2 D T) 0 0) /\ (band (mgt D 0) (mequals T (mplus D 2))) => success;

prove [10,10] in main : apr;
