必須課題5

自然数 num (≧ 0)が与えられたとき，0 から num までの和を求める関数 sum1, sum2 を以下の2通りで作成せよ．
ただし，負の数が(関数の引数として)入力された場合の例外処理も記述すること．

関数名 sum1 sum2
形 式 int sum1( int n ) int sum2( int n )
引 数 整数値 n 整数値 n
返り値 整数値 整数値
機 能 0 から n までの総和を返す．
仕 様
  • 再帰呼び出しは使用しない．
  • n が負のときは 0 を返す．
  • 再帰呼び出しを使用する．
  • n が負のときは 0 を返す．

作成プログラムの仕様

本課題のプログラムとして作成する ex005.c は以下を満たすようにすること．

  • 関数 sum1 の定義はメイン関数の前に記述．
  • 関数 sum2 の定義はメイン関数の後に記述．
  • メイン関数は標準入力から整数１つを受け取り，その整数を sum1, sum2 に与えたときの結果を表示する．

ファイル名ex005.c
標準入力整数
機能 入力された整数に対する sum1, sum2 の結果をそれぞれ表示する

動作例

% gcc -o ex005 ex005.c
% ex005
自然数を入力してください: 10
sum1 による総和: 55
sum2 による総和: 55
% ex005
自然数を入力してください: -3
sum1 による総和: 0
sum2 による総和: 0
%

総和関数の再帰定義

自然数上の総和関数 Sum は以下のように再帰的に定義される． 

Sum( 0 ) = 0
Sum( n ) = n + Sum( n-1 )    [ n > 0 のとき]

本課題では整数上の総和関数であるので， sum2 を定義する際には負の数の場合を補うこと!!
