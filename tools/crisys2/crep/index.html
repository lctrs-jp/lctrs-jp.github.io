Notes

  • The programs shown in this site were provided from students in actual classes for C programming exercises, who agreed to provide the reports for this database.
  • The programs are divided to some groups where the members are structurally very similar.
  • The file name consists of the example name x, year yyyy, a serial number n, and the group name α: x-yyyy-n-α.c.
  • Serial numbers used in this database are not complete, i.e., there are some missing numbers due to some reason, e.g., the corresponding program could not be compiled.
  • Comments written in Japanese were automatically removed from the programs.
  • Strings printed by printf were translated into English by hand if containing Japanese characters.

Own functions for strings

Specification

Todo:
Implement strcpy, strcat, strlen (and strcmp) yourself.
Prototype:
  • void mystrcpy(char trg[], char src[]);
  • void mystrcat(char trg[], char src[]);
  • int mystrlen(char trg[]);
  • int mystrcmp(char trg[], char src[]);
Definition:
  • mystrlenis the same as strlen.
  • mystrcpy and mystrcat are almost the same as strcpy and strcat, resp., but they don't return any value.
  • The requirement to the returned value of mystrcmp is not shown explicitly at the exercise, and thus, it may return -1, 0, or 1.
Note:
  • This exercise was shown before learning pointers.
